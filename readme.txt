На данный момент диск размечается только в BTRFS
На диске должен быть тип GPT и создано минимум два 
раздела ->  boot  (512М если одно ядро) тип EFI Syst
root (остальное  пространство) тип  Linux filesystem
root пространство будет размечено в BTRFS и созданы
subvolume необходимые для системы
можно по своему усмотрению создать ещё разделы

Процес установки поэтапный

1. Загружаемся с устоновочной флешки.
 Подготавливаем загрузку установочных скриптов из github:

pacman-key --init

pacman -Syy

pacman -Sy git

2. Загружем установочные скрипты:

git clone https://github.com/sergeukostrukov/myinstall.git


cd myinstall/

3. Делаем файл исполняемым

chmod +x ./archmini.sh

4. Запускаем установочный скрипт

./archmini.sh

Скрипт установит минимальную конфигурацию без графической надстройки

5.

Устанавливаем графическую надстройку (если нужно)

Скрипт kde.sh запускать с правами root  !!!!!!!!!!!!!

тдт обычным пользователем sudo ./kde.sh

Делаем файл исполняемым

chmod +x ./kde.sh

Запускаем установку

./kde.sh

Скрипт установит рабочий стол KDE Plasma Блютуз Timeshift

6.

Скрипт yayandex.sh устанавливает yay  и yandex browser

 пользователя имеющего права sudo
